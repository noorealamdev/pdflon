## Laravel quick-start Pack with Profile controller and admin theme
- Clone the repo
- Run: php artisan key:generate
- Run: composer install, composer update and npm install
- Run: php artisan migrate
- http://127.0.0.1:8000/register

- http://127.0.0.1:8000/dashboard
- email: admin@admin.com
- pass: 12345678


## Deployment
- npm install
- npm run dev
- npm run prod
